# Otel-Rezervasyon-Sitesi
PHP ve MySQL kullanılarak geliştirilmiş, dinamik içerik yönetimine sahip full-stack otel rezervasyon sistemi. Kullanıcıların oda müsaitlik durumunu kontrol edebildiği ve rezervasyon oluşturabildiği, yöneticilerin ise oda ve rezervasyon verilerini yönetebildiği bir platformdur
# Otel Rezervasyon Sistemi | Full-Stack PHP & MySQL

* **Backend:** PHP (Süreç yönetimi ve form işleme)
* **Veritabanı:** MySQL (İlişkisel veritabanı tasarımı ve SQL sorguları)
* **Frontend:** HTML5 & CSS3 (Dinamik ve responsive sayfa yapısı)
* **Özellikler:** Kullanıcı girişi, dinamik oda listeleme ve SQL tabanlı rezervasyon kayıt sistemi.
